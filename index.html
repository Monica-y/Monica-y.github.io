使用电话网和有线电视网进行数据传输

电话网络

1、电话网络由本地局，中继局和地区局组成。
2、数据传输和信令任务在现代电话网中是分开的：一个网络用来执行数据传输，另一个用来执行信令任务。但不是每处都存在独立的物理链路，两个网络可用系统部分中同一链路的不同通道。

3、上图中SCP为服务控制点。且电话机与信号点之间链路是两个网络公用的。可以印证2的说法。

电话调制解调器

1、传统的电话线路可以传送频率范围是300到3300Hz的信号（整个频率范围用于传输语音），用于数据传输的电话线路的频率范围是600到3000Hz。
2、v.92
能自己调整速度；
如果噪声允许，上传数据速率可以是48kbps。在上传过程中，模拟信号仍然在交换站进行采样。在这个方向上，量化噪声引入信号降低了信噪比，将上载的数据速率限制为33.6kbps（v.90版本）。
下载速率是56kbps。在下载过程中没有采样。信号不受量化噪声的影响，不受香农极限容量原理的限制。
线路有呼叫等待服务，在有电话拨入时，调制解调器可以中断因特网连接。

数据用户线路（DSL）技术

1、ADSL技术是非对称数字用户线路。与56K调制解调器（上文中v.90版本）相似之处是，ADSL的下行（从因特网到住户）速率（比特率）比上行（从住户到因特网）速率要高。与56K调制解调器的非对称性的不同之处是，ADSL是将本地回路的可用带宽进行了不均衡的划分。因此，ADSL是专门针对住宅网络用户而设计的非对称通信技术，它不适用于商业应用。
2、现有的本地回路能够处理的带宽可以高达1.1MHz。
3、ADSL是一种自适应技术，系统使用的数据速率依据本地回路的线路状况而定。
4、ADSL的标准调制技术称为离散多音频技术（DMT），它将正交振幅调制（QAM）和频分多路复用（FDM）结合在一起。一般情况下，将1.104MHz的可用带宽划分为256个通道，每个通道使用的带宽是4.312kHz。
5、高数字用户线路（HDSL）是为了替代T-1线路（1.544Mbps）而设计的。它使用2B1Q编码，使用双绞线实现全双工传输。
6、对称用户线路（SDSL）是HDSL一种双绞线型，它提供全双工对称通信。
7、甚高速数字用户线路（VDSL）是与ADSL类似的一种替代实现技术，适于短距离（300到1800m）的通信，使用同轴电缆、光纤或者双绞线，采用DMT调制。

有线电视网络

1、传统有线电视网中的通信是单向的。
2、第二代有线网络称为光纤同轴电缆混合（HFC）网络。从有线电视局到称为光线节点的分线盒之间的传输介质是光纤，而从光线节点穿过相邻住户并到达房屋内部的线路仍然使用同轴电缆。
3、HFC有线电视网络中的通信可以是双向的。
4、在HFC系统中，将同轴电缆（光纤节点到用户住宅之间）的带宽划分为三个波段：视频、下行数据和上行数据。
下行数据波段：使用64-QAM（也可能是256-QAM）调试技术进行调制。理论上的下行数据速率是30Mbps，但是，由于有线调制解调器是通过10Base-T电缆连接到计算机上的，因此数据速率限制为10Mbps。
上行数据波段：使用QPSK调制技术进行调制。理论上的上行数据速率是12Mbps。
5、为了使用有线网络进行数据传输，需要两种关键设备：有线调制解调器（CM）和有线调制解调器系统（CMTS）。

纠错与检错

一、差错的类型

1、单个位差错：在给定的数据单元（例如一个字节、字符或是分组）中仅有一位发生从1到0或从0到1的变化。
2、突发性差错：在数据单元中有两位或是更多位发生1到0或0到1的变化。
注意：突发性差错并不意味着差错在连续位上出现。突发性差错的长度按从第一个差错位到最后一个差错位的长度计算。其间的某些位也可能未产生差错。

二、前向纠错和重传

1、向前纠错：接收方通过使用冗余位尝试推测报文的方法。如果差错的个数很小，这是可能的。
2、重传：接收方检测出有差错发生并要求发送方重新发送报文的技术。

三、模运算

1、在模运算中，只使用有限范围中的正整数。它的上限称为模数N。然后使用0到N-1的正整数。
2、在模N系统中，如果数字大于N，则除以N并取余数。如果它是负数，需要加上N的倍数使其变为正整数。
3、模运算中的加法和减法是简单的。在同一列上的两个数字相加没有进位，在同一列上的两个数字相减没有借位。
4、XOR运算：如果两个位相同，XOR运算的结果是0；如果不同，则结果是1.

四、块编码

在块编码中，我们把报文划分成块，每个块有k位，称为数据字，并增加r个冗余位使其长度变为n = k + r,形成n位的块称为码字。

五、最小汉明距离

1、两个字的汉明距离是对应位不同的数量。对两个字进行异或操作并计算1的个数，就是汉明距离。
2、最小汉明距离是一组字中所有可能对的最小汉明距离。
3、三个参数：码字长度n，数据字长度k，最小汉明距离d(min)。
4、如果s个差错发生在传输中，发送的码字和接收到的码字间的汉明距离是s。如果我们的编码能检测出最多s个差错，那么两个有效编码间的最小汉明距离必须是s+1，这样接收到的码字才不会与有效码字匹配。
5、为了保证纠正所有情况下最多t个差错，块编码中的最小汉明距离d(min) = 2t + 1。

六、线性块编码

1、在线性块编码中，任两个有效码字的异或(XOR)生成另一个有效码字。
2、线性编码的最小汉明距离是具有最少1的个数的非0有效码字中1的个数。
3、简单奇偶校验码(码字中1的个数是偶数的情况)：n = k + 1、d(min) = 2的单个位检错编码，它不能纠正任何差错。

传输介质

一、总述

1、传输介质实际上位于物理层以下并直接由物理层控制。可以说，传输介质位于第0层。
2、按照电信中的用途，传输介质可以分为两大类：有向的和无向的。有向介质包括双绞线、同轴电缆和光缆，无向介质通常是空气。

二、双绞线

1、线路中的一条用于将信号传送到接收方，另一条仅用作接地参考点。接收方使用两条线路的电平之间的差值。
2、绞合可以使两条线路同等地受到外部影响（噪声或者串扰）。这意味着，接收方不会通过计算两者之间的差值接收到有害信号。
3、3类双绞线的数据速率为10Mbps，5类双绞线的数据速率为100Mbps，5E类双绞线的数据速率为125Mbps。

三、同轴电缆

同轴电缆与双绞线相比，可以传输更高频率范围的信号。

四、光缆

1、传播模式：
多模：阶跃折射率模式和渐变折射率模式
单模：单模使用阶跃折射率光纤并且使用聚焦效果更好的光源，这样可以把光束限制在更小的角度范围内，几乎接近于水平，这样光束的传播基本上是水平的，从而可以忽略延迟，所有的光束同时到达目的端。
2、光纤的优点与缺点：
优点：
1、高带宽
2、信号衰减小
3、无电磁干扰
4、抗腐蚀材料
5、重量轻
6、不易被窃听
缺点：
1、安装/维护
2、方向性
3、成本

五、无线通信

1、无向信号的传播方法有地表传播（低于2MHz）、天空传播（2MHz到30MHz）和视线传播（30MHz以上）。
2、无线传输可以分为三大组：无线电波、微波和红外波。
3、无线电波（3kHz到1GHz）用于多播通信，如收音机、电视以及寻呼机系统。
4、微波（1到300GHz）用于单播通信，如移动电话、卫星网络和无限局域网。
5、红外波（300GHz到400THz）可以在封闭区域用于短距离通信，使用视线传播。

C++中指针栈模板的建立

方法

使用指针栈的方法之一是让调用程序提供一个指针数组，其中每个指针都是指向不同的字符串。把这些指针放在栈中是有意义的，因为每个指针都将指向不同的字符串。

注意

1、创建不同的指针式调用程序的指责，而不是栈的职责。栈的任务是管理指针，而不是创建指针。
2、对于指针栈来说，这里的Type类型一定是一种指针类型，如const char*之类。

3、栈的成员变量items实际是一个二级指针，它用来记录存放栈元素的内存空间的地址，就像是一个数组中的数组名一样，只不过一般的数组中存放的是数据，这个数组中存放的是指针而已。

4、由于items类似于数组名，所以在析构函数中，处理方式就跟数组一样。

洛谷 P1102 A-B 数对

给出一串数以及一个数字C，要求计算出所有A - B = C 的数对的个数（不同位置的数字一样的数对算不同的数对）。原题

借鉴的思路

这个思路的核心就是将A - B = C转换位A - C = B。
1、在输入的时候使用map来统计每个数出现的次数。
2、通过将每一个数都减去C来将A转换成B，如此，只需要统计符合要求的B的次数就可以得到这个题目的答案。
3、对map进行一次遍历，得到所有符合要求的B的出现次数的累加和，即为正确答案。原因：1中统计的是一长串数中每个数字出现的频率，如果没有在这里出现的数字其频率自然为0，在2中将每个A所要求的B求出来，并将它的值直接赋给A，于是就可以拿着这个想要查找到的目标数组去map表里查，最终就很容易得到了符合要求的B的个数。

我的思路

不管采用什么样的顺序，最深的坑总是最花费最多的天数才能被填满，所以每一次总是当前从最深的坑下手，然后以当前最深的坑为起点，向前向后遍历寻找符合要求的最长填坑区间，找出区间后将对应区间的坑深度减一。思路是没有漏洞的，但是洛谷提交后三个点会超时，在千方百计优化以后，仍然会超时，是思路及本身时间复杂度的问题。就是想不到题解的思路，只有七十分，果然模拟不能AC这道题。
